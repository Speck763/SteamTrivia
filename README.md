# SteamTrivia

Trivia web interactiva basada en tu perfil de Steam.

## Desarrollo local

```bash
npm install
npm run dev
```

## Deploy a Vercel

1. Conectar el repo a [Vercel](https://vercel.com)
2. Configurar la variable de entorno `STEAM_API_KEY` en Vercel Dashboard
3. Deploy automático en cada push

## Variables de entorno

| Variable | Descripción |
|---|---|
| `STEAM_API_KEY` | Tu Steam Web API Key ([obtener aquí](https://steamcommunity.com/dev/apikey)) |
